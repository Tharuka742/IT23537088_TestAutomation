# IT23537088_TestAutomation

Automated test suite for evaluating the accuracy of a Singlish to Sinhala transliteration system using Python and Playwright.

---

## 📌 Assignment Context

This project was developed as part of the **IT3040 – ITPM Semester 1 (Year 3) Assignment 1**.

### 🎯 Objective
The objective of this assignment is to evaluate how accurately the application below converts **chat-style Singlish input into Sinhala output**:

https://www.pixelssuite.com/chat-translator

The focus is ONLY on:
- Chat Sinhala transliteration accuracy

The following are **out of scope**:
- Standard Sinhala transliteration  
- Backend/API testing  
- Performance or security testing  

---

## 🚀 Features
- Automated UI testing using Playwright
- Excel-driven test case execution
- Automatic result logging (PASS / FAIL)
- Supports multiple Singlish input scenarios
- Retry mechanism for stable output capture
- Dynamic column detection in Excel

---

## 🛠 Technologies Used
- Python 3.12
- Playwright (sync API)
- OpenPyXL
- Google Chrome / Chromium

---

## 📋 Prerequisites

Ensure the following are installed:

- Python 3.11 or 3.12  
- pip  
- Google Chrome (recommended)

👉 Playwright will install Chromium if needed
